이 섹션에서는 **메타데이터 파일 규칙**에 대해서 다룹니다. 파일 기반 메타데이터는 라우트 세그먼트에 특별한 메타데이터 파일을 만듦으로써 선언할 수 있습니다.

파일 기반 메타데이터는 `opengraph-image.jpg`처럼 정적 파일을 만들어 선언하거나, `opengraph-image.js`처럼 메타데이터를 동적으로 생성하는 코드 파일을 만들어 선언할 수 있습니다.

Next.js는 선언된 메타데이터 파일들을 자동으로 배포하고 (프로덕션에서는 캐싱에 사용되는 해시를 포함), head 요소에 메타데이터 정보를 알맞게 반영합니다. 반영되는 메타데이터 정보에는 배포된 에셋들의 URL, 파일 타입, 이미지 크기 등이 있습니다.
