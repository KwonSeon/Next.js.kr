---
title: trailingSlash
description: 후행 슬래시와 상관없이 Next.js 페이지를 사용할 수 있도록 설정합니다.
---

기본적으로 Next.js는 후행 슬래시가 있는 URL을 후행 슬래시가 없는 URL로 리다이렉트합니다. 예를 들어 `/about/`은 `/about`으로 리다이렉트합니다. 이 기본동작과 반대로 후행 슬래시가 없는 URL을 후행 슬래시가 있는 URL로 리다이렉트하도록 설정할 수 있습니다.

`next.config.js`를 열고 `trailingSlash`설정을 추가하세요.

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

이 설정으로 `/about`과 같은 URL이 `/about/`으로 리다이렉트합니다.
