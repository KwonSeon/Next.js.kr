[Turbopack](https://turbo.build/pack) (베타) Rust로 작성되었고 Next.js에 내장되어 있으며 JavaScript 및 TypeScript에 최적화된 점진적 번들러입니다.

## 사용법

Turbopack은 더 빠른 로컬 개발을 위해 `pages` 및 `app` 디렉터리의 Next.js에서 사용할 수 있습니다. Turbopack을 활성화하려면 Next.js 개발 서버를 실행할 때 `--turbo` 플래그를 사용하십시오.

```json filename="package.json" highlight={3}
{
  "scripts": {
    "dev": "next dev --turbo",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```

## 지원되는 기능

Turbopack에 현재 지원되는 기능을 자세히 알아보려면 [문서](https://turbo.build/pack/docs/features)를 참조하세요.

## 지원되지 않는 기능

Turbopack은 현재 `next dev`만 지원하고 `next build`는 지원하지 않습니다. Turbopack이 점점 안정화 단계에 접어들면서, 빌드를 지원하기 위한 작업도 함께 진행되고 있습니다.
